# Task Dashboard App

A modular frontend dashboard built with HTML, CSS, and Vanilla JavaScript.  
It demonstrates scalable UI architecture, state management, and persistent client-side storage using localStorage.

---

## 🚀 Features

- Add, edit, delete, and toggle todos
- Advanced filtering system (All / Active / Completed)
- Task analytics (basic insights & counts)
- Notification feedback system
- Persistent data using localStorage

---

## 🧠 Architecture

The project is structured using a **feature-based modular architecture**:

- **Core Layer** → state management, events, storage, utilities  
- **Widgets Layer** → independent feature modules (todos, weather, quotes, analytics)  
- **Layout Layer** → UI structure and composition  
- **Styles Layer** → global and modular styling  

This separation ensures scalability and maintainability as features grow.
